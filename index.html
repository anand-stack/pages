C++ implementation of Parser Classes

It is common in bioinformatics to work with various different file formats . It common and it's a challange also. Currently, various libraries are available for achieving such a task. But it's never be easier for new programmer to understand these libraries. Here a simple program in c++ 


            #include <iostream>
            #include <fstream>
            #include <string>

            using namespace std;

            int main(int argc, char** argv)
            {
              if(argc < 2) cout << "Usage :" << argv[0] << endl;
              else
              {
                ifstream file(argv[1]);
                while(file)
                {
                  string line;
                  getline(file,line);
                  if(!line.empty())
                  {
                    cout << line << endl;
                  }
                }
              }
            }
